# Search

AdamRMS has a powerful global search engine that lets you quickly find items across your entire business from a single search box.

## Using Search

1. Click the **search icon** in the top navigation bar, or navigate to the search page directly.
2. Type your keywords into the search box.
3. Press **Enter** or click the **Search** button.

Results are displayed in a list, with each result showing its title, an excerpt of matching content, and a coloured tag indicating the type of result.

## What You Can Search

The global search covers the following types of content:

| Type | Tag Colour | What's Searched |
|------|-----------|----------------|
| **Projects** | Pink | Project names and details |
| **Locations** | Indigo | Location names and addresses |
| **Clients** | Light Blue | Client names and details |
| **CMS Pages** | Primary (Blue) | Page content and titles |
| **Users** | Secondary (Grey) | User names |

Clicking on a search result takes you directly to that item's page.

:::tip
For searching **assets** specifically, use the dedicated [Asset Search](../assets/finding-assets) page, which provides advanced filtering by category, manufacturer, group, availability dates, and more. A link to the asset search is provided directly on the search results page.
:::

## Pagination

Search results are returned in pages of 25 results by default. If your search returns more than 25 results, use the pagination controls at the top of the results list to navigate between pages.

The search page also shows how many total results were found and how long the search took.

## Tips for Effective Searching

- **Be specific**: Use distinctive keywords to narrow down results. For example, search for a client's name rather than a generic term.
- **Use asset search for equipment**: The global search does not include assets -- use the [dedicated asset search](../assets/finding-assets) for finding equipment.
- **Check different categories**: Results are mixed together from all types. Look at the coloured tags to identify what type each result is.
