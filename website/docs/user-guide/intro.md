# Welcome to the AdamRMS User Guide

AdamRMS is a free, open source advanced Rental Management System designed for Theatre, AV & Broadcast. It helps you manage equipment, projects, crew, finances, and locations -- all from one platform.

This guide covers everything you need to know to use AdamRMS effectively, from setting up your account to managing complex multi-project events.
